Propriétés et méthodes de l’objet global String

L’objet String gère les chaînes de caractères.

Le constructeur String() possède deux propriétés et une trentaine de méthodes.

Comme nous l’avons vu précédemment, nous n’utiliserons pas la fonction constructeur de cet objet pour créer de nouveaux objets de type string : on préfèrera en effet utiliser des valeurs primitives qui sont plus performantes et avec lesquelles on va également pouvoir utiliser les propriétés et méthodes définies dans le constructeur.

Les propriétés de l’objet String

Le constructeur String() ne possède que deux propriétés :
- une propriété length et, bien évidemment,
- une propriété prototype comme tout objet.

La propriété length va nous permettre d’obtenir la longueur d’une chaine de caractères. Cette longueur est exprimée en points de code (appelées « codets ») sur la base du format UTF-16. La plupart des caractères comptent pour une unité ou un codet mais certains caractères spéciaux vont être représentés par deux codets.
Attention donc : la propriété length ne renverra pas toujours une valeur égale au nombre de caractères présents dans la chaine. 

        // Propriété length
        let ch = "Bonjour";
        document.write("Nbr car de Bonjour : " + ch.length);

Les méthodes de l’objet String

Le constructeur String() dispose d’une trentaine de méthodes. Nous allons étudier celles qui me semblent les plus utiles ici.

  1. La méthode include() : permet de déterminer si une chaine de caractères est incluse dans une autre - Retourne true ou false
  2. Les méthodes startsWith() et endsWith() : permettent de déterminer si une chaine de caractères commence ou se termine par une autre chaine - Retourne true ou false
  3. La méthode substring() : permet de retourner une sous-chaîne de la chaîne courante à partir d’un indice de départ et d'un indice de fin facultatif.
  4. Les méthodes indexOf() et lastIndexOf() : permettent de retourner l'indice de la première ou de la dernière occurrence de la valeur spécifiée dans une chaîne.
  5. La méthode slice() : permet de retourner une copie d'une partie d'un tableau dans un nouveau tableau.
  6. La méthode replace() : permet de retourner une nouvelle chaîne de caractères dans laquelle certaines occurrences d'une chaîne de caractères spécifiée sont remplacées par une autre chaîne de caractères.
  7. Les méthodes toUpperCase() et toLowerCase() : permettent de convertir une chaine de caractères en majuscules ou en minuscules.
  8. La méthode trim() : permet de supprimer les espaces en début et en fin de chaîne. Elle retourne une nouvelle chaîne. 
            
        let ch = "Hello World !";

        // Méthode includes() 
        document.write("includes() : " + ch.includes("World"));

        // Méthode startsWith()
        document.write("startsWith() : " + ch.startsWith("Hello"));

        // Méthode endsWith()
        document.write("endsWith() : " + ch.endsWith("World"));

        //La méthode substring()
        document.write("substring() : " + ch.substring(0, 5));

        // La méthode indexOf() 
        document.write("indexOf() : " + ch.indexOf("W"));        
        document.write("indexOf() : " + ch.indexOf("World"));

        // La méthode lastIndexOf()
        document.write("lastIndexOf() : " + ch.lastIndexOf("l"));

        // La méthode slice() 
        document.write("slice() : " + ch.slice(0, 5));

        // La méthode replace()        
        document.write("replace() : " + ch.replace("World", "Mars"));

        // Les méthodes toUpperCase() et toLowerCase() 
        document.write("toUpperCase() : " + ch.toUpperCase());
        document.write("toLowerCase() : " + ch.toLowerCase());

        // La méthode trim()
        document.write("trim() : " + ch.trim());

Il existe d’autres méthodes,telles que les méthodes match, matchAll() et search() qui vont trouver tout leur intérêt lorsqu’elles vont être utilisées avec des expressions régulières ou encore la méthode split() qui nécessite de connaitre les tableaux pour être utilisée correctement.
