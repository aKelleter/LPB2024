Exercices en JavaScript

Quelques exercices...

1. Définir et appeler une fonction sans arguments

Dans cet exercice vous devez :
- créer une fonction nommée getNum qui renvoie le nombre 42
- appeler la fonction getNum et stocker le résultat dans une variable nommée num
- afficher le contenu de la variable num dans la console
- afficher le contenu de la variable num dans la page HTML

2. Définir et appeler une fonction avec arguments

Dans cet exercice vous devez :
- créez une fonction nommée divideByTwo(arg1, arg2) qui prend 2 arguments (arg1 et arg2)
- la fonction doit diviser par deux (arg1 + arg2) si l'addition de (arg1 + arg2) est un nombre pair sinon on retourne un message
- appeler la fonction divideByTwo avec deux paramètres de votre choix (faire plusieurs essais) et stocker le résultat dans une variable nommée resultat
- afficher le contenu de la variable resultat dans la console
- afficher le contenu de la variable resultat dans la page HTML

3. Définir et appeler une fonction avec arguments sur les chaînes de caractères

Dans cet exercice vous devez :
- créez une fonction nommée isToLong(arg1, arg2) qui prend en argument 1 (arg1) une chaîne de caractères et en argument 2 (arg2) un nombre entier
- la fonction isToLong() retourne vrai (true) si la longueur de la chaîne ne dépasse pas le nombre passé dans l'arg2 et false dans le cas contraire
- appeler la fonction divideByTwo avec deux paramètres de votre choix (faire plusieurs essais) et stocker le résultat dans une variable nommée resultat
- afficher le contenu de la variable resultat dans la console
- afficher le contenu de la variable resultat dans la page HTML

Utilisez arg1.length pour connaitre la taille chaîne de caractères arg1

4. Définir et appeler une fonction - Tests imbriqués

- Créez une fonction nommée funEval(note) qui :
- prend une note sur 20 en argument
- renvoie Hideux si la note est entre 0 et 7 exclus
- renvoie Moche si la note entre 7 et 10 exclus
- renvoie Nice si la note entre 10 et 15 exclus
- renvoie Extra si la note entre 15 et 20 inclus
- renvoie BUG si la note n'est pas correcte
- appeler la fonction funEval() avec un paramètre de votre choix (faire plusieurs essais) et stocker le résultat dans une variable nommée resultat
- afficher le contenu de la variable resultat dans la console
- afficher le contenu de la variable resultat dans la page HTML

5. Créer et afficher et utiliser 2 objets Personnage() - Fonction constructeur

- Créez deux objets avec une fonction constructeur que vous nommerez : Personnage
- Avec les propriétés suivantes : nom, force vie, degats (placez des valeurs, au choix, de type nombre dans vos propriétés)
- Avec les méthodes suivantes: attaque(perso) et guerison()
- La méthode attaque(perso) doit soustraire les dégats du personnage attaqué à sa vie
- La méthode guerison() doit ajouter 10 points de vie au personnage
- Créer deux personnage avec votre fonction constructeur
- Afficher les propriétés des personnages dans la console
- Afficher les propriétés des personnages dans la page HTML
- Faites les s'attaquer entre eux et se soigner
- Ensuite, à nouveau :
- Afficher les propriétés des personnages dans la console
- Afficher les propriétés des personnages dans la page HTML
- Et observer les changements
