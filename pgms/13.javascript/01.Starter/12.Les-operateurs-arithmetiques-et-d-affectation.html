Les opérateurs d'arithmétiques et d'affectation

Un opérateur est un symbole qui va être utilisé pour effectuer certaines actions notamment sur les variables et leurs valeurs.
Par exemple, l’opérateur * va nous permettre de multiplier les valeurs de deux variables, tandis que l’opérateur = va nous permettre d’affecter une valeur à une variable.
Il existe différents types d’opérateurs qui vont nous servir à réaliser des opérations de types différents. Les plus fréquemment utilisés sont :

• Les opérateurs arithmétiques ;
• Les opérateurs d’affectation / d’assignation ;
• Les opérateurs de comparaison ;
• Les opérateurs d’incrémentation et décrémentation ;
• Les opérateurs logiques ;
• L’opérateur de concaténation ;
• L’opérateur ternaire ;
• l’opérateur virgule.

Pour le moment, nous allons nous concentrer particulièrement sur les opérateurs arithmétiques et d’affectation ou d’assignation. Nous étudierons les autres par la suite lorsqu’on devra les utiliser dans certaines structures JavaScript.

Les opérateurs arithmétiques

Les opérateurs arithmétiques vont nous permettre d’effectuer des opérations mathématiques sur des valeurs numériques.
Les opérateurs arithmétiques les plus courants sont :

• L’addition (+) ;
• La soustraction (-) ;
• La multiplication (*) ;
• La division (/) ;
• Le modulo (%) ;
• L’élévation à la puissance (**).

Pour utiliser un opérateur arithmétique, il suffit de placer le symbole de l’opérateur entre deux valeurs numériques.
Par exemple, pour additionner deux nombres, il suffit d’utiliser l’opérateur + entre ces deux nombres.

x = 5;
y = 3;
z = 4;

a = x + 1;
b = x + y;
c = x - y;
d = x * y;
e = x / y;
f = x % y;
g = x ** y;

Les opérateurs d’affectation

Les opérateurs d’affectation vont nous permettre d’affecter une valeur à une variable.
Les opérateurs d’affectation les plus courants sont :

• L’opérateur d’affectation simple (=) ;
• L’opérateur d’affectation d’addition = additionne puis affecte (+=) ;
• L’opérateur d’affectation de soustraction = soustrait puis affecte (-=) ;
• L’opérateur d’affectation de multiplication = multiplie puis affecte (*=) ;
• L’opérateur d’affectation de division (/=) = divise puis affecte;
• L’opérateur d’affectation de modulo = effectue le modulo puis affecte (%=) ;
• L’opérateur d’affectation d’élévation à la puissance = effectue l'élévation de puissance puis affecte (**=).

Pour utiliser un opérateur d’affectation, il suffit de placer le symbole de l’opérateur entre le nom de la variable et la valeur à affecter.
Par exemple, pour affecter la valeur 5 à la variable x, il suffit d’utiliser l’opérateur = entre x et 5.

t = 5;
u = 3;
v = 4;

t += 1;
u -= 1;
v *= 2;
t /= 2;
u %= 2;
v **= 2;

Ici, vous devez bien comprendre que les opérateurs d’affectation combinés nous permettent d’effectuer deux types d’opérations à la suite. Dans l’exemple ci-dessus, on réalise des opérations arithmétiques puis d’affectation. Ainsi, l’opérateur += par exemple va nous permettre d’ajouter une valeur à la valeur contenue dans une variable puis d’affecter le résultat dans cette même variable. La nouvelle valeur va ainsi écraser la valeur précédente.
